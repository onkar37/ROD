Description :

	The project GREP searches the named input FILEs (or standard input 
        if no files are named, or the file name - is given) for lines 
        containing a match to the given PATTERN.  By default, grep prints 
	the matching lines. The project grep is developed for fast searching.
		Grep have options like printing line numbar, search the directory 
	recursivly,print the lines after pattern matching and etc.
